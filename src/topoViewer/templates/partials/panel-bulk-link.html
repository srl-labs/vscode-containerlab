Bulk Link Devices

Enter patterns to connect multiple devices at once. Source Pattern selects the starting nodes and Target Pattern selects destinations for each source. You can use wildcards (*, ?, #) or regular expressions, and reuse capture groups from the source in the target with placeholders like $1 or $<name>.

For example, matching leaf* to spine* links every leaf to all spines, while leaf?# to spine$1 connects each numbered leaf to the spine with the same index.
